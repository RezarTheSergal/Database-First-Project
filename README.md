## Стек
- Язык: Python
- GUI: PySide6
- БД: PostgreSQL + SQLAlchemy
- Юнит-тестирование: pytest

## Локальный запуск

В корне проекта:

```bash
python -m venv env
source env/scripts/activate
pip install -r requirements.txt
```
